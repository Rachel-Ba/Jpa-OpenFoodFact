Le master contient le code initial de l'application avec très peu de classes:
 - les entités métier (Produit, Categorie, Marque, etc) dans le package entity
 - la classe principale IntegrationApp exécutable
 - une classe d'exception
 - une classe StringUtils qui propose des méthodes de traitements de chaine de 
   caractères pour supprimer des informations indésirables dans la liste des ingrédients,
   des allergènes et des additifs
 - une classe Convertisseur qui propose des méthodes simples de conversion de String
  en Integer et de String en Double
